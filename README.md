# DataNova AI

Analyze • Visualize • Predict

## Overview

DataNova AI is an AI-powered data analysis platform that enables users to upload CSV, Excel, JSON, or connect SQL databases, perform automated exploratory data analysis, generate intelligent visualizations, ask questions in natural language, train machine learning models, generate Python and SQL code on demand, manage analysis tasks, and create professional reports—all within a single workspace.

## Features

- CSV Analysis
- Excel Analysis
- JSON Analysis
- SQL Database Support
- AI Chat
- Automatic EDA
- Data Visualization
- Machine Learning
- Task Management
- Report Generation
- Code Generation
